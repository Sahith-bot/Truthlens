# TruthLens — AI News Verifier

TruthLens is a real-time AI fact-checking and news verification application. It analyzes news articles using heuristics and cross-references them against global sources to verify accuracy.

## Features

- **Real-Time Fact-Checking**: Instantly analyze articles and statements.
- **Verdict Summarization**: Visual metrics representing trust, bias, and correctness levels.
- **Global Source Checking**: Verifies against an extensive collection of global news and reference sources.
- **Multi-Language Support**: Accessible layout across multiple languages.

## Deployment

This is a single-page web application. It can be easily deployed to Vercel, Netlify, or GitHub Pages by simply serving the `index.html` file.
